# Руководство по установке и использованию

## PostgreSQL

### Папка `./postgres`

Для управления PostgreSQL доступны следующие варианты:

- **WSL/Linux**: Используйте shell-скрипт
  ```bash
  ./pg.sh {{command}}
  ```

- **Windows**: Установите npm и используйте
  ```bash
  npm run {{command}}
  ```

### Доступные команды

| Команда   | Описание              |
|-----------|----------------------|
| `start`   | Запуск контейнера    |
| `stop`    | Остановка контейнера |
| `restart` | Перезапуск контейнера|
| `status`  | Статус контейнера    |

> **⚠️ Важно**: Перед запуском скопируйте файл `.env.example`, переименуйте в `.env` и настройте переменные по необходимости.

## Backend

### Системные требования

- **Node.js**: >= 20.0.0
- **npm**: >= 9.0.0

### Первоначальная настройка

Для первого запуска выполните:
```bash
npm run firsttime
```

> **⚠️ Важно**: Перед запуском скопируйте файл `.env.example`, переименуйте в `.env` и настройте переменные по необходимости.

### npm команды

| Команда      | Описание                                                    |
|--------------|-------------------------------------------------------------|
| `start`      | Запуск готового билда                                       |
| `watch`      | Режим разработки с автоматической пересборкой при изменениях|
| `build`      | Создание production билда                                   |
| `copyfiles`  | Копирование необходимых файлов в директорию билда          |
